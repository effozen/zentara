# 빠른 시작

## 기본 사용법

먼저, `ZentaraInput` 컴포넌트를 가져와서 사용합니다:

```jsx copy
import { ZentaraInput } from '@zentara/core';

export default function App() {
  const [value, setValue] = useState('');

  return (
    <ZentaraInput
      value={value}
      onChange={setValue}
      placeholder='입력해주세요...'
    />
  );
}
```

## 플러그인 추가하기

Zentara는 플러그인을 통해 더 강력해집니다. 다음은 suggestions 플러그인을 사용하는 예시입니다:

```jsx copy
import { ZentaraInput } from '@zentara/core';
import { suggestionsPlugin } from '@zentara/plugin-suggestions';

export default function App() {
  const [value, setValue] = useState('');

  return (
    <ZentaraInput
      value={value}
      onChange={setValue}
      plugins={[
        {
          plugin: suggestionsPlugin,
          config: {
            rules: [
              {
                triggers: [':'],
                suggestions: Object.keys(emojiMap),
              },
            ],
          },
        },
      ]}
    />
  );
}
```

## 스타일링

className이나 style props를 사용하여 외관을 커스터마이징할 수 있습니다

```jsx
<ZentaraInput
  value={value}
  onChange={setValue}
  className='my-custom-input'
  style={{ width: '100%', padding: '8px' }}
/>
```

## 에러 상태

에러 메시지를 표시하려면 `error` prop을 사용합니다:

```jsx
<ZentaraInput value={value} onChange={setValue} error='필수 입력 항목입니다' />
```
