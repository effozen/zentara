# ZentaraInput

Zentara의 메인 컴포넌트입니다. 플러그인으로 확장 가능한 커스터마이징 가능한 입력 필드를 제공합니다.

## Props

```typescript
interface ZentaraInputProps {
  /** 입력 필드의 초기값 */
  value?: string;

  /** 값이 변경될 때 호출되는 콜백 */
  onChange?: (value: string) => void;

  /** 입력 필드를 확장하는 플러그인 배열 */
  plugins?: Plugin[];

  /** CSS 클래스 이름 */
  className?: string;

  /** 인라인 스타일 */
  style?: CSSProperties;

  /** 플레이스홀더 텍스트 */
  placeholder?: string;

  /** 입력 필드 아래에 표시될 에러 메시지 */
  error?: string;
}
```

## 기본 사용법

```jsx
import { ZentaraInput } from '@zentara/core';

function MyComponent() {
  const [value, setValue] = useState('');

  return (
    <ZentaraInput
      value={value}
      onChange={setValue}
      placeholder='입력해주세요...'
    />
  );
}
```

## 플러그인 사용

참고: 여러 플러그인이 커스텀 입력 렌더러(`renderInput`)를 제공하는 경우, 첫 번째 플러그인만 사용되며 콘솔에 경고가 표시됩니다.

```jsx
import { ZentaraInput } from '@zentara/core';
import { suggestionsPlugin } from '@zentara/plugin-suggestions';

function MyComponent() {
  return (
    <ZentaraInput
      plugins={[
        suggestionsPlugin({
          rules: [
            {
              trigger: '@',
              items: ['user1', 'user2', 'user3'],
              transform: (item) => `@${item} `,
            },
          ],
        }),
      ]}
    />
  );
}
```

## 에러 상태

```jsx
<ZentaraInput value={value} onChange={setValue} error='필수 입력 항목입니다' />
```
